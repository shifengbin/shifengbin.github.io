利用Minecraft学习Python

2025-12-03 12:18:36    |    1 分钟    |    34 字    |    Fengbin

环境步骤:

  1. 新的树梅派系统已经不自带Minecraft了，想要安装最简单的方法是先安装Pi-Apps

  2. 安装后需要在Games里找到Minecraft，点击安装

  3. 安装mcpi库来控制Minecraft 执行 sudo pip3 install mcpi

开始使用python控制：

  1. 随便找一个顺手的编辑器，创建一个py文件

  2. 编写python代码，例子后面举例

  3. 执行python代码，用命令也好，用编辑其自带功能也好，都可以

例子：

import mcpi import minecraft

#本地游戏创建
mc = minecraft.Minecraft.create()
#远程游戏创建
#mc = minecraft.Minecraft.create("192.168.1.1", 4711)

mc.postToChat("123123") #向屏幕发送文字
