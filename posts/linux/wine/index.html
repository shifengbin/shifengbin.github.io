使用flatpak安装wine后执行winecfg配置wine时出现乱码可以使用命令安装字体解决

sudo apt-get install fonts-wqy-zenhei fonts-wqy-microhei

# 安装基础组件
winetricks corefonts cjkfonts  # 自动安装中文支持

# 设置中文环境
winetricks setwinver=win10
winetricks fakechinese  # 模拟中文系统
